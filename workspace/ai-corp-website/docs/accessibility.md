# アクセシビリティチェックリスト

WCAG 2.1 AA準拠のためのチェックリスト

## パーセプティブル（知覚可能）

### 1.1 テキストの代替

- [ ] **画像**: すべての非装飾画像に説明的なalt属性を設定
- [ ] **装飾画像**: `alt=""` または `role="presentation"` を使用
- [ ] **SVG**: titleタグまたはaria-labelで説明を追加
- [ ] **アイコン**: アイコンの目的を説明するテキストを隣接配置

### 1.2 時間ベースのメディア

- [ ] **動画**: キャプションまたは字幕を提供
- [ ] **音声コンテンツ**: トランスクリプトを提供
- [ ] **自動再生**: 自動再生メディアを回避、または停止/一時停止ボタンを提供

### 1.3 適応可能

- [ ] **情報と構造**: HTMLセマンティクスを正しく使用（heading, nav, main, article, section等）
- [ ] **意味のある順序**: コードの順序が視覚的な順序と一致
- [ ] **入力エラー**: エラーメッセージを説明的にする

### 1.4 判別可能

- [ ] **色のみの区別**: 色だけで情報を伝えない（テキストやアイコンも併用）
- [ ] **コントラスト**: テキストと背景のコントラスト比 4.5:1 以上（大型テキストは 3:1 以上）
- [ ] **テキストのサイズ**: テキストは200%まで拡大可能で、機能が損なわれない
- [ ] **背景音**: 前景の音声コンテンツより少なくとも20dB低くする

## オペラブル（操作可能）

### 2.1 キーボード操作

- [ ] **キーボードアクセス**: すべての機能をキーボードのみで操作可能
- [ ] **キーボードトラップ**: キーボードフォーカスが抜け出せない状態を回避
- [ ] **ショートカット**: キーボードショートカットを無効化または代替手段を提供

### 2.2 十分な時間

- [ ] **タイムアウト**: タイムアウト前に警告し、延長オプションを提供
- [ ] **一時停止**: 自動スクロールや更新を停止/一時停止可能

### 2.3 てんかんと身体的反応

- [ ] **点滅コンテンツ**: 1秒間に3回以上点滅するコンテンツを回避

### 2.4 ナビゲーション可能

- [ ] **フォーカス順序**: フォーカスの順序が論理的で意味がある
- [ ] **リンクの目的**: リンクテキストが目的を明確に説明（「ここをクリック」は避ける）
- [ ] **見出し**: 見出しを使用してコンテンツを構成
- [ ] **フォーカス_visible**: フォーカスインジケーターが視認可能

## アンダースタンダブル（理解可能）

### 3.1 読みやすさ

- [ ] **言語の宣言**: `lang` 属性でページの言語を宣言（例：`<html lang="ja">`）
- [ ] **言語の一部**: 他言語の部分に `lang` 属性を適用

### 3.2 予測可能

- [ ] **一貫したナビゲーション**: 同じ種類のナビゲーションは一貫した順序
- [ ] **一貫した識別**: 同じ機能のコンポーネントは一貫したラベル
- [ ] **コンテキスト変更**: ユーザーの要求なしにコンテキストを変更しない

### 3.3 入力支援

- [ ] **エラーの指摘**: 入力エラーを明確に特定し、説明する
- [ ] **ラベルや指示**: 入力フィールドにラベルまたは指示を提供
- [ ] **エラー修正**: エラー修正のための提案を提供

## ロバスト（堅牢）

### 4.1 互換性

- [ ] **マークアップ**: HTML仕様に従い、正しく閉じタグを使用
- [ ] **名前・役割・値**: ARIA属性を正しく使用
- [ ] **通知**: ステータスメッセージを適切に通知

## ARIA属性チェックリスト

### ランドマークロール

- [ ] `role="banner"` (header)
- [ ] `role="navigation"` (nav)
- [ ] `role="main"` (main)
- [ ] `role="complementary"` (aside)
- [ ] `role="contentinfo"` (footer)
- [ ] `role="search"` (検索フォーム)

### ラベル付け

- [ ] `aria-label`: 要素にアクセシビリティ名を提供
- [ ] `aria-labelledby`: 他の要素を参照してラベル付け
- [ ] `aria-describedby`: 説明テキストを参照

### 状態・プロパティ

- [ ] `aria-expanded`: 展開/折りたたみ状態
- [ ] `aria-current`: 現在のページまたは項目
- [ ] `aria-hidden`: 補助技術から隠す要素
- [ ] `aria-pressed`: トグルボタンの状態
- [ ] `aria-selected`: 選択状態

### ライブリージョン

- [ ] `aria-live="polite"`: 重要な更新通知
- [ ] `aria-live="assertive"`: 緊急の通知
- [ ] `aria-atomic="true"`: 更新領域全体を通知

## フォームチェックリスト

- [ ] すべてのinputに対応するlabelがある
- [ ] required属性を使用して必須項目を示す
- [ ] aria-invalid="true"でエラー状態を示す
- [ ] placeholderではなくlabelを使用
- [ ] エラーメッセージは視覚的かつスクリーンリーダーで認識可能

## キーボードナビゲーションチェック

- [ ] Tabキーですべてのインタラクティブ要素にアクセス可能
- [ ] Shift+Tabで逆順にナビゲート可能
- [ ] Enter/Spaceでボタンやリンクをアクティベート
- [ ] 矢印キーでメニューやタブを操作可能
- [ ] Escapeキーでモーダルやドロップダウンを閉じる

## コントラストチェック

### テキストコントラスト（AA基準）

- [ ] 通常テキスト: 4.5:1 以上
- [ ] 大型テキスト（18pt以上または太字14pt以上）: 3:1 以上
- [ ] UIコンポーネントとグラフィックオブジェクト: 3:1 以上

### 推奨カラーコンビネーション

```css
/* 良いコントラストの例 */
:root {
  --text-primary: #333333;      /* コントラスト比: 12.6:1 */
  --text-secondary: #666666;    /* コントラスト比: 7.5:1 */
  --background-primary: #ffffff;
  --background-secondary: #f5f5f5;
  --accent-color: #667eea;      /* 白背景: 4.5:1 */
  --link-color: #0066cc;        /* 白背景: 7.4:1 */
}
```
