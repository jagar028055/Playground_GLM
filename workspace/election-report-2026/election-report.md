### 1-2. 政治的背景

#### 政権移動と政党再編

2025年末から2026年始めにかけて、日本の政治構図は劇的な変化を遂げた：

1. **高市政権の誕生**（2025年10月）
   - 自民党総裁選で高市早苗氏が党総裁に就任
   - 第102代内閣総理大臣に就任

2. **公明党の連立離脱**（2025年12月）
   - 長年続いた自公連立の解消
   - 公明党が野党に転換

3. **日本維新の会との連立**（2026年1月）
   - 自民党が維新と新たな連立政権を樹立
   - 安保政策などで政策接近

4. **中道改革連合の結成**（2026年1月）
   - 立憲民主党（148議席）と公明党（24議席）が合流
   - 計172人の衆議院議員が参加する新党を結成

このような短期間での政権交代と政党再編は、戦後日本政治史上例を見ない急展開となった。

## 2. 各政党の公約・政策比較

> **担当**: Worker2

本章では、主要10政党の公約・政策を分野別に比較分析する。各政党の立場を客観的に整理し、有権者の判断材料を提供する。

### 2-1. 経済・財政政策

#### 2-1-1. 消費税政策

今回の衆院選で最大の争点の一つとなっている消費税政策。長引く物価高を背景に、与野党すべての政党が何らかの減税策を掲げている。

##### 政党別比較表

| 政党 | 基本方針 | 対象・期間 | 財源 |
|------|----------|-----------|------|
| **自民党** | 食料品2年間減税 | 食料品のみ、2年間 | 税外収入、補助金、租税特別措置見直し |
| **中道改革連合** | 食料品恒久的ゼロ | 食料品のみ、恒久化（2026年秋〜） | 政府系ファンド創設、国の基金活用 |
| **維新** | 食料品2年間減税 | 食料品のみ、2年間 | 「国民会議」で検討 |
| **国民民主** | 一律5%減税 | 全品目、賃金上昇率が物価+2%達成まで | 明記せず |
| **共産** | 直ちに5%に減税→将来的に廃止 | 全品目、段階的廃止 | 大企業・富裕層への課税強化 |
| **れいわ** | 即時廃止 | 全品目、即時廃止 | 法人税引き上げ検討 |
| **参政** | 廃止・ゼロ | 全品目、即時廃止 | 積極財政 |
| **社民** | 即時ゼロ | 全品目、即時廃止 | 防衛費削減・法人税・所得税累進性強化 |
| **保守** | 食料品恒久的ゼロ | 食料品のみ、恒久化 | 省庁・事業・海外拠出金の大胆削減 |
| **みらい** | 維持（現行10%） | 変更なし | 社会保険料削減優先 |

##### 政策ポジション分析

**減税の3つのパターン**:

1. **廃止派**: 共産・れいわ・参政・社民
   - 全品目の即時廃止または段階的廃止を掲げ
   - 財源は「大企業・富裕層への課税強化」や「防衛費削減」
   - 社会的弱者への配慮を重視

2. **期間限定減税派**: 自民・維新
   - 「2年間限定」「食料品のみ」と条件付き
   - 与党として財政規律を維持しつつ、即効性のある支援
   - 財源の明確性に課題あり

3. **恒久減税派**: 中道改革・保守
   - 食料品の恒久ゼロ化を掲げ
   - 財源は「政府系ファンド」や「行財政削減」

4. **維持・社会保険料削減派**: みらい
   - 消費税率維持、社会保険料削減を優先

**財源の不透明性**:
多くの政党で減税の財源が明確でない。自民党は「税外収入・補助金・租税特別措置見直し」、中道改革は「政府系ファンド・基金活用」としているが、具体性に欠ける。一方、共産・社民は「大企業・富裕層への課税強化」を明記。

#### 2-1-2. 物価高対策・家計支援

| 政党 | 主な対策 | 給付付き税額控除 | 社会保険料 |
|------|----------|------------------|------------|
| **自民** | 食料品消費税軽減 | 〇導入明記 | - |
| **中道改革** | 食料品消費税ゼロ、給付付き税額控除 | 〇導入明記 | - |
| **維新** | 食料品消費税軽減 | 〇導入明記 | - |
| **国民民主** | 一律5%減税 | ×社会保険料還付制度導入 | 〇還付制度創設 |
| **共産** | 最低賃金1500円、内部留保課税 | - | - |
| **れいわ** | 現金10万円一律給付、季節ごとにインフレ対策 | - | - |
| **参政** | 国民負担率45.8%→35%に抑制 | - | 〇削減 |
| **社民** | あなたの税金はあなたに | - | - |

**給付付き税額控除**: 自民・中道改革・維新の3党が導入を掲げている。中低所得者や現役世代の手取りを増やす仕組みで、給付と減税を組み合わせた政策。

**国民民主党の代替案**: 「社会保険料還付制度」の導入を掲げる。給付付き税額控除ではなく、社会保険料の還付による家計支援。

**れいわ新選組の特徴**: 「現金10万円一律給付」を消費税減税までの「つなぎ」として掲げる他、「季節ごとにインフレ対策給付」を実施。

#### 2-1-3. 金融政策・経済成長戦略

| 政党 | 金融政策の立ち位置 | 成長戦略の特徴 | 具体的投資分野 |

| **維新** | 経済を動かす | 技術力強化 | 半導体など戦略的物資の研究開発支援 |

**成長戦略の特徴**:
- **自民党**: 「責任ある積極財政」を掲げ、AI・半導体など17の戦略分野に集中投資
- **中道改革**: グリーン・ローカル・ライフ・デジタルの4分野に重点投資
- **維新**: 半導体など戦略的物資の研究開発支援で技術力強化
- **国民民主**: デジタル・環境分野で「ハイパー償却税制」導入
- **れいわ**: 10年間で200兆円のグリーン投資

#### 2-1-4. 経済・財政政策のまとめ

**類似点**:
- 与野党を問わず、物価高対応として何らかの減税策を掲げ
- 自民・中道改革・維新で「給付付き税額控除」導入で合意
- 成長重視の立場が与党系を中心に主流

**相違点**:
- 消費税の扱いで「廃止」vs「減税」vs「維持」で大きな違い
- 財源の明確性に大きな格差（大企業課税 vs 経済成長 vs 不明）
- 社会保険料の扱い（削減優先 vs 還付制度 vs 無言及）

##### 原発政策の3つのグループ

**推進派**: 自民・維新・国民民主・みらい・参政（5党）
- 再稼働推進、次世代革新炉の開発・建設
- 電力安定供給とエネルギー安全保障を重視
- 参政は「次世代原子力・核融合など民間投資だけでは賄えない分野には国として投資」

**中間派**: 中道改革（1党）
- 「将来的に原発に依存しない社会を目指しつつ、安全性が確実に確認され、実効性のある避難計画があり、地元の合意が得られた原発の再稼働」
- 条件付きで容認
- 立憲民主党の従来方針（「2050年原発ゼロ」）から転換

**反対派**: 共産・れいわ・社民（3党）
- 共産: 「原発の再稼働、新増設に反対し、原発ゼロの日本をめざす」
- れいわ: 「原発は即時、廃止。国が事業者から買い上げ、最先端の技術を用いて慎重に廃炉」
- 社民: 「再生可能エネルギーの普及で脱原発をすすめる」

**特徴的な転換**:
中道改革連合の結成に伴い、立憲民主党は「2050年までのできるだけ早い時期に原発ゼロ」という前回のマニフェストをあっさりと転換。立憲の幹部も含む少なからぬ議員は、与党・民主党として福島第一原発事故を経験し、2012年野田政権時に大規模な「国民的議論」を経て「2030年代に原発ゼロを目指す」という政策決定を行っている経緯があり、今回の転換は大きな政策変更となっている。

- 内閣支持率70.8%は高水準
- しかし「この時期の選挙」への批判も根強い
- 投票率次第で与党への逆風も可能性

### 5-2. 選挙結果のシナリオ分析

序盤情勢調査では自民党がリードしているが、有権者の投票行動次第でいくつかのシナリオが考えられる。

#### シナリオA：自民単独過半数（233議席以上）

##### 状況

自民党が単独で過半数を獲得し、高市政権の信任が決定的になる。

##### 要因

1. 高市早苗首相の圧倒的な支持率（70.8%）を組織票に転化
2. 中道改革連合が伸び悩み、野党票が分散
3. 無党派層の「政権継続」志向
4. 133選挙区での自民・中道・参政の競合で自民が保守票を固定化

##### 連立の行方

- 維新との連立継続か、単独政権への移行か
- 単独政権の場合、維新が野党転落の可能性
- または維新を閣外協力に位置づけ

##### 政治構図

```
自民党（単独過半数）
  ↓
安定政権基盤
  ↓
次段階の政策推進（憲法改正、防衛費増額等）
```

##### 確率

**高**（序盤情勢調査で自民リード、「青木の法則」でも与党圧倒有利）

#### シナリオB：自民・維新で過半数（自民200-232 + 維新20-40）

##### 状況

現状の連立体制を維持し、与党で過半数を確保。

##### 要因

1. 自民党は単独過半数に届かず
2. 維新は現有議席（41議席）を維持・微増
3. 与党内対決（85選挙区）で保守票が分裂

##### 連立の行方

- 自民・維新の連立継続
- 両党の政策調整が課題
- 防衛費・憲法改正で一致、経済政策で微妙な立場の違い

##### 政治構図

```
自民党（200-232）+ 維新（20-40）= 過半数確保
  ↓
現状の連立維持
  ↓
政策調整の課題
```

##### 確率

**中**（最も現実的と見られている）

#### シナリオC：与党過半数割れ（自民・維新合計で232以下）

##### 状況

与党が過半数を割り込み、高市政権の退場へ。

##### 要因

1. 高市首相の支持率が投票所に結びつかず
2. 中道改革連合が予想以上に躍進（150議席以上？）
3. 無党派層の「政権交代」志向
4. 若年層の投票率が上昇し野党有利に傾く
5. 133選挙区で自民・中道・参政の競合で自民が大敗

##### 野党連立政権の可能性

**中道改革連合を軸にした政権構想**:

```
中道改革連合（150？）+ 国民民主党（30？）+ 共産・社民（部分的協力）
  ↓
過半数確保（233議席以上）
  ↓
野党連立政権樹立
```

**課題**:
- 中道改革連合と国民民主党の政策協調
- 共産党との共闘の是非（安保政策での温度差）
- れいわ新選組・参政党を含めるか

##### 政治構図

```
与党過半数割れ
  ↓
高市政権の退場
  ↓
中道改革連合を軸に野党連立政権？
  ↓
政党再編の加速度化
```

##### 確率

**低**（但し無党派層の動向次第では可能性）

#### シナリオD：ねじれ国会（衆院で与党過半数、参院で野党過半数）

##### 状況

衆院選では与党が過半数を獲得するが、参院では野党が過半数を維持。

##### 要因

1. 2025年参院選での結果（中道改革連合が参院で第一党？）
2. 衆院と参院での投票行動の違い
3. 参院の「良識の府」としての意思表示

##### 政治の停滞

- 衆院で可決された法案が参院で否決・修正
- 両院協議会の開催
- 衆院の優越議決（2/3の賛成が必要）
- 政治の停滞が続く

##### 政治構図

```
衆院：与党過半数
参院：野党過半数
  ↓
ねじれ国会
  ↓
政治の停滞
  ↓
早期衆院解散の可能性
```

##### 確率

**中**（2025年参院選の結果次第）

### 5-3. 今後の政治情勢展望

今回の衆院選結果は、日本の政治構図にどのような変化をもたらすのか。4つの視点から展望する。

#### 5-3-1. 政党再編の加速

今回の選挙結果次第で、さらなる政党再編が加速する可能性がある。

##### パターン1：中道改革連合が躍進（150議席以上）

- 野党第一党としての地位を確立
- 国民民主党の吸収合併？
- 維新からの流出議員の取り込み
- 「保守・リベラルの二大政党」体制へ移行

##### パターン2：自民単独過半数（233議席以上）

- 維新の野党転落・解党の可能性
- 一部維新議員の自民入党
- 中道改革連合の分裂（公明党系 vs 立憲系）
- 「自民一強」体制の強化

##### パターン3：与党過半数割れ

- 高市首相の退場
- 自民党の内部的対立（後継総裁選）
- 中道改革連合を軸にした野党連立政権樹立
- 維新の連立与党入りも視野

##### パターン4：現状維持（自民・維新連立で過半数）

- 大きな政党再編はなし
- 維新は連立与党として定着
- 中道改革連合は野党第一党として政策競争
- 「多党化の時代」が定着

#### 5-3-2. 二大政党制の崩壊と多党化の定着

戦後日本の政治は、「55年体制」下での自民党一優位、その後の「自社さ連立」「自公連立」と、実質的な二大政党制（自民 vs 民主/立憲）が続いた。

しかし、今回は以下の要因で二大政党制が崩壊し、多党化が定着する。

##### 要因1：政党再編の混沌

```
従来：自民 vs 民主/立憲（二大政党）
  ↓
今回：自民・維新 vs 中道改革連合 vs 国民民主 vs 参政党・共産など（多党化）
```

##### 要因2：無党派層の増加

- 無党派層は有権者の30-40%を占める
- 政党への帰属意識の低下
- 個人（候補者）重視の投票行動

##### 要因3：イシュー別の政党支持

- 経済政策：自民・維新
- 環境政策：共産・れいわ
- 外国人政策：保守・参政
- 1つの政党で全政策を満たす有権者は減少

##### 多党化の定着による影響

**連立政権が常態化**:
- 単独過半数を獲得する政党は稀に
- 連立政権の形成・解消が頻繁に
- 政権不安定化のリスク

**政策決定の複雑化**:
- 連立与党内での政策調整が必須
- 大連立（自民×中道など）の可能性も
- 政治の停滞が懸念される

#### 5-3-3. 有権者の投票行動の変化

今回の選挙は、有権者の投票行動にも変化をもたらしている。

##### 変化1：政党より個人（候補者）重視

- 政党への帰属意識の低下
- 候補者の人物本位・政策本位の投票
- SNSでの候補者発信の重要性増大

##### 変化2：無党派層の鍵を握る

- 無党派層は有権者の30-40%
- 与野党の競合区では無党派層の振るい方が勝敗を決
- 各政党は無党派層への訴求を強化

##### 変化3：若年層の投票行動

- 若年層の投票意向61%は高齢層の77%を大きく下回る
- しかしSNSを通じた政治関心は高まる傾向
- 若年層の投票率上昇で野党有利の可能性

##### 変化4：イシュー投票の増加

- 単一の政党支持ではなく、イシュー（政策）別の投票
- 例：経済政策は自民、環境政策はれいわ、など
- 比例代表と小選挙区で投票先が分かれる現象も

#### 5-3-4. 今後の課題と展望

今回の衆院選の結果、日本の政治は以下の課題に直面する。

##### 課題1：政治の停滞

- 多党化による連立政権の常態化
- 政策決定の複雑化・遅延
- 衆院と参院の「ねじれ」の可能性

##### 課題2：有権者の政治不信

- 短期間での総選挙への批判
- 政党再編の混沌に対する不満
- 政治とカネの問題の未解決

##### 課題3：政策決定の困難

- 物価高対策・エネルギー政策・外交安保など課題山積
- 連立与党内での政策調整が必須
- 大きな合意形成が困難

##### 課題4：国際社会との関係

- G7で唯一「多党化」が進む国
- 政権不安定化による外交力の低下
- 中国・北朝鮮等の情勢悪化への対応

##### 展望

**短期的（1年以内）**:
- 選挙結果による政治構図の変化
- 新政権の発足・政策決定
- 連立与党内での政策調整

**中期的（1-3年）**:
- 政党再編の加速度化
- 新たな政党の誕生・既存政党の解体
- 二大政党制への回帰か、多党化の定着か

**長期的（3-5年）**:
- 日本の政治システムの根本的見直し
- 憲法改正の議論の進展
- 有権者の政治参加意識の向上

## まとめ

第51回衆議院議員総選挙（2026年2月8日投開票）は、戦後日本政治史上例を見ない激動の選挙となる。

**選挙の特徴**:
1. 前回からわずか1年3ヶ月での総選挙（16日間の超短期決戦）
2. 政党再編の混沌（自公解消→自維連立、立憲公明→中道改革連合）
3. 133選挙区で自民・中道・参政が競合する複雑な構図

**最大の争点**:
1. 新与党連立（自民・維新）の正当性
2. 中道改革連合の「第三極」としての定着
3. 物価高対策の具体性（消費税減税、財源の確保）
4. 短期的政権運営の是非

**情勢分析**:
- 高市早苗首相の支持率70.8%を追風に、序盤情勢では自民党がリード
- しかし中道改革連合は伸び悩み、野党統一候補の擁立も進まず
- 与党内対決（自民VS維新）で保守票の分裂が避けられない
- 無党派層の動向が鍵を握る

**選挙結果のシナリオ**:
- シナリオA：自民単独過半数（高）
- シナリオB：自民・維新で過半数（中）
- シナリオC：与党過半数割れ（低）
- シナリオD：ねじれ国会（中）

**今後の展望**:
- 政党再編の加速度化
- 二大政党制の崩壊と多党化の定着
- 連立政権の常態化による政治の停滞
- 有権者の投票行動の変化（政党→個人重視）

2月8日の投開票結果は、日本の政治構図をどう変革させるのか、政党再編をどう加速させるのか、歴史的な総選挙として記憶されることだろう。
