## 5. 政策ポジションマップ

### 5-1. 経済・財政政策の軸

```
【消費税: 廃止 ← → 維持】
廃止派: 共産・れいわ・参政・社民
減税派: 自民・維新・中道改革・国民民主
維持派: 保守・みらい

【財源: 増税 ← → 成長・削減】
増税派: 共産・社民（大企業・富裕層課税）
成長派: 自民・維新・保守（経済成長）
基金活用: 中道改革
```

### 5-2. エネルギー政策の軸

```
【原発: 推進 ← → 廃止】
推進: 自民・維新・国民民主・みらい・参政
条件付き: 中道改革
反対: 共産・れいわ・社民

【気候: 野心的 ← → 消極的】
野心的: 共産（25点）・れいわ（25点）
中間: 自民・維新・中道改革（1-5点）
消極的: 参政（-5点）・保守（-3点）
```

### 5-3. 安保・憲法の軸

```
【防衛: 強化 ← → 縮小】
強化: 自民・維新・国民民主
現状維持: 中道改革
縮小: れいわ

【憲法改正: 推進 ← → 反対】
推進: 自民・維新
議論: 中道改革
反対: 共産・れいわ
```

## 6. 類似点・相違点のまとめ

### 6-1. 経済政策の類似点

1. **物価高対応**: 多くの政党が何らかの減税策を掲げ
2. **給付付き税額控除**: 自民・中道改革・維新で共通
3. **成長重視**: 与党系を中心に成長戦略を掲げ

### 6-2. 経済政策の相違点

1. **消費税の扱い**: 「廃止」vs「減税」vs「維持」で大きな違い
2. **財源の明確性**: 大企業課税（共産・社民）vs 経済成長（自民・維新）vs 不明（多数）
3. **社会保険料**: 削減優先（保守・みらい）vs 還付制度（国民民主）

### 6-3. エネルギー政策の類似点

1. **カーボンニュートラル**: 多くの政党が2050年達成を掲げ（参政除く）
2. **再エネ導入**: 全党が拡大方針（参政は縮小）

### 6-4. エネルギー政策の相違点

1. **原発**: 推進4党 vs 反対3党 vs 条件付き1党で二分
2. **気候目標**: 共産・れいわの野心的目標 vs 他党の現状追認
3. **石炭火力**: 2030年全廃（共産・れいわ）vs 混焼推進（自民・維新）

### 6-5. 安保政策の類似点

1. **防衛産業基盤**: 与党系で防衛産業強化で共通
2. **スパイ防止**: 維新・参政で法制化を掲げ

### 6-6. 安保政策の相違点

1. **憲法改正**: 推進（自民・維新）vs 反対（共産・れいわ）
2. **防衛費**: 増額（与党）vs 削減・中止（れいわ）

## 7. 政党グループ別の特徴

### 7-1. 連立与党（自民・維新）

**経済**: 消費税2年間減税、給付付き税額控除、成長戦略
**エネルギー**: 原発推進、次世代革新炉、脱石炭目標なし
**安保**: 防衛費増額、装備品輸出拡大、憲法改正推進

### 7-2. 中道改革連合（公明・立憲合流）

**経済**: 食料品消費税恒久ゼロ、給付付き税額控除
**エネルギー**: 将来的に原発依存しない社会、条件付き再稼働
**安保**: 適切な防衛力整備、責任ある改憲論議

**特徴**: 両党の従来政策が混在、原発で立憲の「2050年原発ゼロ」から転換

### 7-3. 気候政策高評価グループ（共産・れいわ・社民）

**経済**: 消費税廃止
**エネルギー**: 脱原発、脱石炭2030年、野心的気候目標
**安保**: 憲法改正反対、防衛費縮小

**特徴**: 社会保障と気候政策を重視、気候ネットワーク評価で25点満点（共産・れいわ）

### 7-4. 原発推進グループ（国民民主・みらい）

**経済**: 国民民主は5%減税、みらいは消費税維持
**エネルギー**: 原発推進、次世代革新炉、2030年25基以上運転（みらい）
**安保**: 自国防衛重視

### 7-5. 規制緩和・成長重視グループ（参政・保守）

**経済**: 参政は消費税廃止、保守は維持
**エネルギー**: 再エネ縮小、脱脱炭素（参政）、再エネ賦課金廃止
**外交**: 外国人規制強化、スパイ防止法（参政）

## 9. 分析のまとめ

### 9-1. 経済政策の特徴

今回の衆院選では、長引く物価高を背景に、全政党が何らかの減税策を掲げているのが特徴。

**消費税政策の3つのパターン:**
1. **廃止派**: 共産・れいわ・参政・社民（全品目廃止・ゼロ）
2. **減税派**: 自民・維新・中道改革・国民民主（期間限定・対象限定）
3. **維持派**: 保守・みらい（社会保険料削減優先）

**財源の不透明性**:
 多くの政党で減税の財源が明確でない。自民は「税外収入・補助金・租税特別措置見直し」、中道改革は「政府系ファンド・基金活用」としているが、具体性に欠ける。一方、共産・社民は「大企業・富裕層への課税強化」を明記。

### 9-2. エネルギー政策の二極化

気候変動対策とエネルギー安全保障のバランスで、政党の立ち位置が明確に分かれている。

**原発推進派の増加**:
 自民・維新・国民民主・みらい・参政と、原発推進を掲げる政党が過半数を占める。背景には、電力安定供給への懸念やエネルギー安全保障の重視がある。

**気候目標の格差**:
- **共産**: 2035年度に75-80%削減（パリ協定1.5℃目標対応）
- **れいわ**: 2030年に70%削減
- **他党**: 現行目標維持または記載なし

参政は「パリ協定離脱」「炭素目標撤回」を掲げ、国際的潮流に逆行。

**中道改革の転換**:
 立憲民主党が2025年参院選で「2050年までのできるだけ早い時期に原発ゼロ」を掲げたが、中道改革では「条件付き再稼働容認」へ転換。公明党との合流で従来の立憲のエネルギー政策が後退。

### 9-3. 安保政策の維持

与党系は防衛強化の維持、野党は憲法改正反対が基本ライン。

**防衛装備品輸出**:
 自民・維新が「5類型撤廃」で足並みそろえ。一方、共産・社民は反対。

**憲法改正**:
 自民・維新が推進、共産・れいわが反対。中道改革は「立憲主義堅持の上で責任ある改憲論議」と中間的立場。

### 9-4. 外国人政策での合流

外国人の土地・不動産取得規制で、自民・維新・国民民主・参政が合流。外国人政策が昨年の参院選に続き争点化。

### 9-5. 政治改革での与党内温度差

自民・維新の連立内でも、企業・団体献金で温度差。
- **維新**: 全面禁止目指し、個人献金促進
- **自民**: 2027年9月末までに結論
