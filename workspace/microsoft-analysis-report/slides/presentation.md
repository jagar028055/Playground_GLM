# 弱気材料（Bear Case）

## 1. 評価額の高さ

- **PER 34倍**: 歴史的にプレミアム
- **10年国債**: 4.2-4.5% vs 株利回り2.9%
- **イールドスプレッド**: -1.3%〜-1.6%

## 2. 成長率の鈍化リスク

- **Azure**: +40%（2022）→ +29%（2024）
- **類似事例**: AWSは成長鈍化で株価横ばい

## 3. AI投資の回収期間不透明

- **インフラ投資**: 年間50-100億ドル
- **過去の教訓**: メタバース（Meta）は収益化困難

# シナリオ分析

## 確率分布と期待収益率

| シナリオ | 確率 | 目標価格 | 収益率 |
|---------|------|---------|--------|
| **ベースケース** | 60% | $626.78 | +20.6% |
| **ブルケース** | 25% | $730.00 | +40.5% |
| **ベアケース** | 15% | $450.00 | -13.4% |

### 期待値の計算

```
期待収益率 = (0.60 × +20.6%) + (0.25 × +40.5%) + (0.15 × -13.4%)
           = +12.4% + +10.1% - 2.0%
           = **+20.5%**
```

**期待収益率+20.5%は買い推奨の根拠**
