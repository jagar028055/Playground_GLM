### 相対評価（Relative Valuation）

#### PEモデル

```
【前提】
- 2025年EPS予想: $13.30
- 業界平均PER: 28倍
- Microsoftプレミアム: +6倍

【計算】
- 目標PER: 34倍
- 目標株価: $13.30 × 34 = $452
- 現在の株価: 約$450-470
→ 現状は適正水準
```

#### PEGモデル

```
【前提】
- PER: 34倍
- 成長率（5年平均）: +14%
- PEGレシオ: 34 / 14 = 2.43

【評価】
- PEG < 1.0: 割安
- 1.0 < PEG < 2.0: やや割高
- PEG > 2.0: 割高
→ Microsoftは「やや割高」 but 成長の質を考慮すれば許容
```

#### EV/EBITDAモデル

```
【前提】
- 2025年EBITDA予想: 約$127B
- EV/EBITDA倍率: 22-24倍（過去5年平均）
- 企業価値: $127B × 23 = $2.9T
- 正味債務: -$33B（ネットキャッシュ）
- 時価総額: $2.9T - (-$0.03T) = $2.93T
- 株価: $2.93T / 7.4B shares = $396
→ 現在はやや割高
```
