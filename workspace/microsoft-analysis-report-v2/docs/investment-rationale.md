## 4. リスク要因と対応策

### 4.1 主要リスク

#### リスク1: Azure成長率の鈍化

**影響度**: 高
**確率**: 中（30-40%）

**現在の状況**:
- Azure成長率: 40% → 39%（1ポイント低下）
- 市場の反応: 決算後12%下落

**監視指標**:
- Azure成長率が35%を割り込むか
- GPU容量制約の解消状況

**対応策**:
- 35%以上維持なら保有継続
- 35%未満なら部分的利益確定検討
- 30%未満なら損切り検討

#### リスク2: AI投資のリターン不透明

**影響度**: 中
**確率**: 中（30%）

**現在の状況**:
- CAPEX: 年間$37.5B（過去最高）
- 市場の懸念: 収益化の時期

**監視指標**:
- AI関連収益の成長率
- Azure AI Servicesの採用数
- CopilotのARPU（1ユーザーあたり収益）

**対応策**:
- 四半期ごとの進捗確認
- 収益化加速の兆しが見えるまで保有
- 1年以内に兆しなければ再評価

#### リスク3: 競合激化

**影響度**: 中
**確率**: 中（40%）

**現在の状況**:
- AWS: 市場シェア31%（依然として最大）
- Google Cloud: 成長率34%（Azureに迫る）
- 価格競争の激化

**監視指標**:
- クラウド市場シェアの変化
- Azureシェアが20%を割り込むか

**対応策**:
- シェア維持なら保有継続
- シェア低下なら部分的利益確定検討
